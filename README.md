# Homework

## Описание

Веб-приложение отображает текущий статус системы управления.

Статус может принимать два значения:
- ON
- OFF

Изменение статуса происходит на стороне сервера через Flask. Текущее состояние сохраняется в файл `status.txt`, поэтому оно не сбрасывается при перезапуске сервера.

---

## Требования

- Python 3
- Flask

---

## Запуск проекта

### Шаг 1. Открыть папку проекта
1. Скачайте проект
2. Откройте VS Code
3. Выберите File → Open Folder
4. Укажите папку проекта

---

### Шаг 2. Открыть терминал
Нажмите:
```
Ctrl + `
```

---

### Шаг 3. Установить Flask
```bash
pip install flask
```

---

### Шаг 4. Запустить сервер
```bash
python main.py
```

После запуска сервер будет доступен:
```
http://localhost:5000/
```

---

## Управление статусом

### Посмотреть статус
```
http://localhost:5000/
```

### Включить ON
```
http://localhost:5000/?status=on
```

### Выключить OFF
```
http://localhost:5000/?status=off
```

Также управление доступно через кнопки на сайте.

---

## Остановка сервера
Нажмите в терминале:
```
Ctrl + C
```

---

```
main.py              - Flask сервер
```
