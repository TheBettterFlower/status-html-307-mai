# Homework

## Описание

Веб-приложение отображает текущий статус механической руки.

Статус может принимать два значения:

* ON
* OFF

Изменение статуса происходит на стороне сервера через Flask. Текущее состояние сохраняется в файл `status.txt`, поэтому сохраняется между запросами и отображается всем пользователям одинаково.

---

## Требования

* Python 3
* Flask

---

## Запуск проекта

### Шаг 1. Открыть папку проекта

1. Скачайте проект.
2. Откройте VS Code.
3. Выберите **File → Open Folder**.
4. Укажите папку проекта.

### Шаг 2. Открыть терминал

Нажмите:

```text
Ctrl + `
```

Откроется встроенный терминал VS Code.

### Шаг 3. Установить Flask

Выполните команду:

```bash
pip install flask
```

Установка требуется только один раз.

### Шаг 4. Запустить сервер

```bash
python main.py
```

После запуска сервер будет доступен по адресу:

```text
http://localhost:5000/
```

---

## Управление статусом

### Просмотр текущего статуса

```text
http://localhost:5000/
```

### Установить статус ON

```text
http://localhost:5000/?status=on
```

### Установить статус OFF

```text
http://localhost:5000/?status=off
```

Статус сохраняется в файл `status.txt`.

---

## Остановка сервера

В терминале VS Code нажмите:

```text
Ctrl + C
```
